What's the Project?

Different machine learning models will be used to analyze Ausralia's rainfall levels between 2011-2016 to predict Australia's rainfall amount for 2017. After utilizing different types of binary and linear models to perform the analysis, the actual 2017 weather report will be compared with the 2017 weather prediction to reveal which model was most effecient at performing long term weather forecasting.

Why are We Doing This?

Negative rainfall has a negative impact on both nature and the economy. Wildfires, disruption of habitats and ecosystems, water shortages, and crop production loss are some results of drought. Accurate long-term weather production is essential for drought prediction. Drought prediction gives Australians time to plan and apply mitigation techniques including water conservation, crop diversification, and product sourcing.

Conclusion

The Random Forest model in both binary and linear forms produced the most accurate results.. Overall, model's precision percentage is conservative, signifying it's predicted rainy days count will be lower than the count of days it actually rained. This can be acceptable if the intention is not to overestimate rainfall. The model's performance can be improved by adding more meteorological features and quality data with missing values.
  • The precision score is a better measure of model performance as opposed to recall
  • For this particular dataset, the precision score is a better measure of performance model
  • Random Forest produced the highest precision and accuracy and balanced accuracy scores compared to other evaluated models
  • Random Forest is robust against overfitting, outliers and large datasets
  • Works well with classification and regression problems
  • Is easier to work with if the dataset has a lot of missing values
  • The model can be difficult to interpret
  • May require higher computational power and training time

Regression Model Analysis

Random Forest regression model produced better results than Linear Regression. The r-square value helps determine how well the dependant variable value is explained, by the determining (independant) variables of the dataset. A higher r-squared value indicates a better fit model and more accurate results. The 32% Random Forest r-squared is on the low side, but can be improved by using data with fewer null values, and adding additional weather features, (ie. wind, sunshine, and humidity).
  • Linear Regression r-squared: 0.16802
  • Random Forest r-squared: 0.32257
